# Horizon

**中文**  
Horizon 是一个用于管理每周任务和长期目标的 Windows 桌面应用。  
它帮助你把“这周要做什么”和“长期想完成什么”放在同一个清晰的系统里，减少任务分散、计划断裂和目标遗忘的问题。

**English**  
Horizon is a Windows desktop app for managing weekly tasks and long-term goals.  
It helps you keep weekly execution and long-term planning in one clear system, so your day-to-day work stays aligned with what matters most over time.

---

## 项目简介 | About

**中文**  
很多任务工具擅长记录临时事项，却不擅长把短期执行和长期规划连接起来。  
Horizon 的目标是提供一个简单、清晰、适合 Windows 桌面使用的工具，让你既能规划每周任务，也能持续追踪长期目标和阶段性进展。

**English**  
Many task tools are good at capturing short-term to-dos, but not at connecting them to long-term goals.  
Horizon aims to provide a simple and focused Windows desktop experience for planning weekly tasks while continuously tracking long-term goals and progress.

---

## 核心目标 | Goals

**中文**
- 管理每周任务
- 追踪长期目标
- 建立任务与目标之间的关联
- 提供清晰、轻量、专注的桌面体验

**English**
- Manage weekly tasks
- Track long-term goals

## 计划功能 | Planned Features

**中文**
- 每周任务创建与编辑
- 长期目标与里程碑管理
- 任务优先级、状态、截止日期管理

**English**
- Create and edit weekly tasks

- Weekly review and progress tracking
- View the relationship between tasks and long-term goals

- Future support for reminders, export, and sync
