# Horizon C1 极夜霓虹 Logo 设计

## 目标

为 Horizon 建立一个酷炫但在 Windows 托盘小尺寸下仍清晰的品牌图标，并统一应用到程序、托盘、安装器、快捷方式和主面板标题区域。

## 核心符号

- 使用用户选定的“任务轨道”构图。
- 深海军蓝圆角方形作为主体底板，画布四角保持透明。
- 青色、电光蓝到紫色的渐变圆形轨道代表长期推进。
- 中心冰白色勾选符号代表任务完成。
- 右上方保留一颗紫色轨道节点，增加运动感和识别点。
- 图标中不出现字母或文字，避免小尺寸不可读。

## 视觉风格

- 配色采用用户选定的 `C1 · 极夜霓虹`。
- 主色：青色 `#57F1FF`、电光蓝 `#4E87FF`、紫色 `#CB58FF`。
- 勾选符号使用偏青的冰白色，保持高对比。
- 背景由深蓝向近黑过渡，不能使用纯黑平底。
- 大尺寸允许柔和外发光、玻璃质感和轻微空间层次。
- 禁止复杂纹理、细小文字、密集粒子和超出安全区的强光晕。

## 混合制作方式

1. 使用图像生成制作 `1024 × 1024` 视觉母版，确定材质、光感和整体气质。
2. 根据母版重绘可控的 SVG 图形，确保轨道、勾和节点边缘稳定。
3. 从矢量源导出应用 PNG 与多尺寸 ICO。
4. `16–32 px` 版本减少光晕、加粗轨道和勾，优先保证托盘辨识度。
5. `48 px` 以上版本保留完整渐变和柔和霓虹质感。

## 文件产物

- `src/Horizon.App/Assets/horizon-logo-master.png`：1024px 视觉母版。
- `src/Horizon.App/Assets/horizon-logo.svg`：可继续编辑的矢量源。
- `src/Horizon.App/Assets/horizon-logo.png`：主界面使用的透明 PNG。
- `src/Horizon.App/Assets/Horizon.ico`：包含 `16、20、24、32、40、48、64、128、256 px` 的 Windows 图标。

## 程序实装

- WPF 项目通过 `ApplicationIcon` 使用 `Horizon.ico`，使 EXE 和 Windows 快捷方式显示新图标。
- 托盘服务继续从当前 EXE 提取图标，因此自动使用同一品牌图标。
- 主面板标题左侧增加紧凑的 Logo 图像，不改变现有标题和任务摘要层级。
- Inno Setup 使用同一 ICO 作为安装程序图标和卸载显示图标。
- 安装器继续生成 x64 自包含版本，版本号更新为 `0.2.1`。

## 尺寸与安全区

- 核心轨道、勾和节点必须完整落在画布中央约 `76%` 的安全区内。
- 圆角底板与画布边缘保留透明间距，避免 Windows 图标蒙版裁切。
- 16px 图标必须仍能分辨“环形轨道＋中心勾”，右上节点可以简化但不能成为噪点。
- 主界面图标显示尺寸控制在 `24–28 px`，不得挤压标题或置顶按钮。

## 验收标准

- EXE、托盘、安装器、开始菜单和桌面快捷方式使用同一 Logo。
- 主面板标题旁显示清晰 Logo，顶部布局无新增遮挡或裁切。
- 16px、24px、32px 和 256px 图标均无明显锯齿、脏边或透明边缘异常。
- 托盘深色与浅色背景下，轨道和勾均可辨识。
- Release 构建、自动化测试、自包含发布和 v0.2.1 安装器生成成功。
- 安装/卸载不会删除 `%LocalAppData%\Horizon\data` 中的任务数据。

## 范围外

- 本次不设计文字版品牌标识、启动动画或开屏页面。
- 不因 Logo 改动重做 Horizon 整体配色体系。
- 不增加动态图标、任务完成角标或托盘状态动画。
